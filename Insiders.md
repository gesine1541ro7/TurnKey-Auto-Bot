# TurnKey Auto Bot — Airdrop (Insiders)

Автоматизация airdrop-задач с консольным интерфейсом в стиле киберпанк.

## Меню и кнопки

| Кнопка | Действие |
|--------|----------|
| **Install Dependencies** | Установка зависимостей из `requirements.txt` (pip install -r requirements.txt) |
| **Settings** | Открытие/редактирование настроек: RPC URL, задержки, путь к кошелькам, лимиты |
| **About** | Вывод информации о проекте, версии и лицензии |
| **Load Wallets** | Загрузка кошельков из файла (wallets.txt или указанный путь) |
| **Run Bot** | Запуск основного цикла бота (симуляция/реальная логика по настройкам) |
| **Export Results** | Экспорт результатов (логи, отчёты) в папку exports |
| **Exit** | Корректный выход из приложения |

## Требования

- Python 3.10+
- Зависимости: см. `requirements.txt`

## Оформление

- Интерфейс в CMD/консоли.
- Стиль: киберпанк (неоновые цвета, рамки, ASCII-логотип).
- Логотип отображается при старте и в заголовке главного меню.

## Структура проекта

```
TurnKey Auto Bot - Airdrop/
├── Insiders.md
├── README.md
├── requirements.txt
├── config.json          # настройки (создаётся через Settings)
├── main.py              # точка входа, меню
├── src/
│   ├── __init__.py
│   ├── installer.py     # установка зависимостей
│   ├── settings.py      # работа с config.json
│   ├── bot.py           # логика бота
│   └── export.py        # экспорт результатов
├── wallets.txt          # опционально: список кошельков
└── exports/             # папка для экспорта
```

## Настройки (Settings)

- `rpc_url` — RPC endpoint (по умолчанию пусто)
- `delay_min`, `delay_max` — задержка между действиями (сек)
- `wallets_path` — путь к файлу с кошельками
- `max_retries` — макс. попыток при ошибке

## Использование

```bash
python main.py
```

Выберите пункт меню цифрой или стрелками (если реализовано).
